Задание по AngularJS

  1. Под каждым полем ввода выводится строка "Available n symbols", где n - параметр MaxLength у каждого из объектов в массиве $scope.params.
  2. Вывести под каждым полем ввода вместо n не параметр MaxLength, а разницу между количеством введенных символов и MaxLength и, если разница отрицательная (т. е. введено больше, чем доступно), выделить эту строку красным цветом.
  3. Введенную строку обрезать до MaxLength нельзя.
  4. Форма должна иметь состояние $invalid, если какое-то из полей не заполнено или превышен лимит введенных символов, как и сейчас.
Available {{p.MaxLength - p.Value.length}} symbols
